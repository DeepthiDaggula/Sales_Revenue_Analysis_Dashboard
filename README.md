# 📊 Sales & Revenue Analysis Dashboard

## 📌 Project Overview

The Sales & Revenue Analysis Dashboard is a Python-based data analytics project developed using Google Colab. It imports sales data, calculates key performance indicators (KPIs), visualizes revenue trends, and identifies top-performing products using charts and business insights.

---

## 🚀 Features

- Import sales data using Pandas
- Calculate total sales and total revenue
- Identify top-performing products
- Visualize revenue using bar charts
- Display sales distribution with pie charts
- Generate business insights

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- Matplotlib
